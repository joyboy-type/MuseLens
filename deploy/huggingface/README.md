# MuseLens public demo

Search a curated image collection with natural-language descriptions. The public Space
runs in fixed-corpus demo mode: searches and transient query images are allowed, while
library imports and persistent visitor uploads are disabled.

Source code and the full local-first version are maintained at
<https://github.com/joyboy-type/MuseLens>.
