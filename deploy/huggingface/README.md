# MuseLens public demo

Search a curated image collection with natural-language descriptions. The public Space
runs in fixed-corpus demo mode: text search, image queries and isolated temporary visitor
galleries are allowed, while persistent library imports are disabled. The 24 fixed demo
images retain their original CC BY 2.0 licenses and are credited in
`demo_assets/ATTRIBUTIONS.md`.

Source code and the full local-first version are maintained at
<https://github.com/joyboy-type/MuseLens>.
